    Play Framework

    Quick start

    Hepek supports Play 2.9 and 3.0 with Scala 3.

    Mill:

    def ivyDeps = Agg(
  ivy"ba.sake::hepek-play-3_0:0.22.0"
)

    Sbt:

    libraryDependencies ++= Seq(
  "ba.sake" %% "hepek-play-3_0" % "0.22.0"
)

    scala-cli:

    //> using dep ba.sake::hepek-play-3_0:0.22.0
